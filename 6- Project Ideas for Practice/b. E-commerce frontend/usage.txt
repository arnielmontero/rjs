Tech Stack Suggestions:

    UI library (Material UI, Chakra UI, Tailwind CSS, or your own styles)

Key Implementation Steps:
    Set up routing for product list, detail, cart, and checkout.
    Create CartContext (or Redux slice) to manage cart state.
    Fetch product data from a mock API.
    Build product list and detail pages.
    Implement cart functionality (add, remove, update).
    Build checkout form with validation.
    Add order confirmation page.
    (Optional): Add authentication and protected routes.
